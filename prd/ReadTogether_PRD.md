## 一、产品概述

### 1.1 产品定位

ReadTogether 是一款面向英语学习者的
**多人在线共享文档平台**，支持学习小组共同阅读、划词翻译、笔记批注与讨论，结合AI辅助学习功能，打造"沉浸式协作阅读"体验。

**一句话概述**：\
\> "让英语学习像一起看剧一样自然 ------
在线协作阅读、实时讨论、AI助学。"

------------------------------------------------------------------------

### 1.2 产品目标

-   支持小组成员在线共享文档，共同阅读学习资料（PDF、文章、教材）\
-   提供同步阅读、划词翻译、批注、讨论等功能，提升学习互动性\
-   用 AI 助手辅助词汇讲解、生成练习题，增强学习效果

------------------------------------------------------------------------

### 1.3 目标用户

  用户类型         特征                     需求
  ---------------- ------------------------ ----------------------------------
  普通英语学习者   自学或组队学习           想要一起读原文、练习口语、做笔记
  学习社群/小组    有共同学习目标的用户群   需要共享资料和进度
  教师/辅导员      组织学生阅读英文资料     希望追踪阅读进度与参与度

------------------------------------------------------------------------

### 1.4 用户场景示例

1️⃣ **学习社群阅读活动**\
\> "一个英语角小组想共同阅读《The
Economist》文章，每天同步阅读一篇，实时划词、讨论。"

2️⃣ **老师布置阅读任务**\
\> "教师上传教材章节，学生在平台上阅读、标注词汇、完成小测。"

3️⃣ **朋友自学组队**\
\> "两三个朋友每晚一起在线读小说、边读边查词，AI 自动解释生词。"

## 二、核心功能设计

### 2.1 功能结构图

**一级功能：** - 文档管理\
- 小组协作\
- 阅读器\
- 批注与翻译\
- AI 助学\
- 讨论互动

------------------------------------------------------------------------

### 2.2 功能详情

#### 🗂 文档管理

  功能       描述
  ---------- ----------------------------------------
  上传文档   支持 PDF、DOCX、TXT、HTML 导入
  文档预览   在线阅读（分页渲染、支持PC和移动端）
  权限设置   上传者可选择：仅自己 / 小组共享 / 公开

#### 👥 小组协作

  功能            描述
  --------------- -------------------------------------------
  创建/加入小组   用户可创建小组并邀请成员（链接或邀请码）
  小组空间        每个小组有独立文件夹和讨论区
  成员权限        拥有者、管理员、成员（不同阅读/上传权限）

#### 📖 同步阅读器

  功能             描述
  ---------------- ------------------------------------------
  实时同步阅读     支持主持人模式：成员可跟随主持人阅读位置
  独立阅读模式     用户可自由阅读、不影响他人
  页码与进度显示   实时显示小组整体阅读进度

#### 🖊 批注与划词

  功能       描述
  ---------- --------------------------------------
  划词翻译   一键显示释义（AI+词典结合）
  添加批注   在句子/段落上添加笔记，支持公开/私密
  收藏生词   将单词加入个人词库
  查看批注   可切换"仅看自己"/"小组批注"视图

#### 💬 讨论互动

  功能            描述
  --------------- ------------------------------------------
  实时讨论区      每个文档右侧有即时聊天窗口
  批注评论        对具体段落发起讨论（类似Notion comment）
  表情/引用支持   简化版富文本输入

#### 🤖 AI 助学模块

  功能       描述
  ---------- ----------------------------------
  AI释义     智能解析划词含义、例句
  段落总结   一键生成段落摘要
  练习生成   基于内容生成选择题/填空题
  学习问答   用户可向AI提问（如语法、背景等）

------------------------------------------------------------------------

## 三、关键交互流程

### 3.1 新用户注册与加入小组

**流程：** 注册 → 创建/加入小组 → 上传文档 → 开始阅读\
**关键点：** 小组邀请码 / 一键加入机制

### 3.2 阅读协作流程

1.  打开共享文档\
2.  选择"同步模式"或"自由模式"\
3.  划词查看翻译 / 添加批注\
4.  在右侧讨论区发起讨论\
5.  AI助手可即时解释单词、生成练习

## 五、非功能需求

  类型     需求
  -------- -------------------------------------------
  性能     支持 50 人同时阅读同一文档；延迟 \< 300ms
  安全     所有文件加密存储；用户权限隔离
  兼容性   优先支持 Web（PC + 移动端）；未来扩展 App
  多语言   UI支持中英切换（初期以中文为主）

## 七、竞争分析（简略）

  产品               特点                                 弱点
  ------------------ ------------------------------------ --------------------------
  Notion             协作强、笔记功能完善                 不适合同步阅读
  Kami               在线批注PDF                          无小组体系，无AI学习功能
  Grammarly / Elsa   AI学习好                             不支持协作阅读
  **ReadTogether**   ✨聚焦英语学习 + 同步协作 + AI助学   差异化明显
